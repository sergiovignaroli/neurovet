🔥 Configurar Firebase

Pegá el objeto firebaseConfig de la consola Firebase:

⚙️ Instrucciones paso a paso para obtener el firebaseConfig
1. Entrá a console.firebase.google.com con tu cuenta Google
2. Click en Agregar proyecto → nombre: NeuroVet → Continuar → Deshabilitar Analytics → Crear proyecto
3. Menú izquierdo → Firestore Database → Crear base de datos → Comenzar en modo de prueba → Elegir región → Habilitar
4. Click en ⚙️ (Configuración del proyecto) en el menú izquierdo
5. Bajá hasta "Tus apps" → click en el ícono </> (Web)
6. Nombre: NeuroVet → click Registrar app
7. Copiá el bloque const firebaseConfig = { ... }
8. Pegá solo la parte { apiKey: "...", ... } en el campo de arriba

⚠️ PASO OBLIGATORIO — Reglas de Firestore:
Sin esto Firebase bloquea todo y no sincroniza nada.
Ir a Firebase → Firestore Database → pestaña Reglas → borrar todo y pegar esto → click Publicar: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

⚠️ TAMBIÉN — Reglas de Firebase Storage (para archivos):
Ir a Firebase → Storage → pestaña Reglas → reemplazar con esto → click Publicar: 
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read, write: if true;
    }
  }
}
Para activar Storage: Firebase → Storage → Empezar → Modo de prueba → Elegir región → Listo.
